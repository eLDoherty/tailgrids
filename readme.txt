== Description ==

This plugin will show the custom guttenberg blocks for tailgrids themes.

== Installation ==

Download the plugin with the zip extension, and run in your wordpress dashboard

e.g.

1. Upload the plugin files to the `/wp-content/plugins/tailgrids` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== FAQS ==

= Can run this plugin on another theme instead of tailgrids? =

Yes, you can run this plugin on another themes. But you should install or get CDN for library and third party that used in this plugin.
You can read below this about what dependencies and library you should bring inside your themes before using this plugins

= Its a safe? =

Its totally safe.
